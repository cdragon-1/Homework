#### 인터넷은 어떻게 작동할까요

- 객체지향 설계 : 현실에 존재하는 모든 것을 속성과 행위로 나태내는 것이다. 여기서 속성은 객체속성(properties), 행위는 메서드(methods)로 구현된다.
- 장고 모델 : 장고 안의 모델은 객체의 특별한 종류이다. 이 모델을 저장하면 그 내용이 데이터베이스에 저장되는 것이 특별한 점이다.
- 데이터베이스 : 데이터들이 모아져 있는 곳이다.


- 서버 구동
python manage.py runserver
=
./manage.py runserver
